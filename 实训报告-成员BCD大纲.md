### 第四章 详细设计与实现

**4.1 开发环境介绍**
- 同通用模板，加上示波器/逻辑分析仪（用于调试 PWM 和单总线时序）

**4.2 电机驱动实现**
- 实现方法：TIM4 双通道 PWM（PB6=左轮, PB7=右轮），频率 1kHz，CCR 范围 0~999；PB12~15 方向控制；`Motor_SetSpeed()` 传入有符号速度值，正=前进，负=后退；`Motor_Forward()` 内置 trim 微调补偿机械偏差
- 实现结果：PWM 波形正常，调速线性度好，差速转向灵活
- 关键代码：`Motor_Init()` PWM 初始化 + `Motor_SetSpeed()` 符号判断 + 方向切换

**4.3 舵机驱动实现**
- 实现方法：TIM2 PWM，频率 50Hz（周期 20ms），脉宽 0.5ms~2.5ms 对应 0°~180° 线性映射。角度钳位保护
- 实现结果：角度控制精度 ±2°，转动平滑
- 关键代码：`Servo_Init()` 50Hz PWM 配置 + `Servo_SetAngle()` 角度→CCR 映射公式

**4.4 超声波测距实现**
- 实现方法：Trig(PA7) 发 10μs 触发脉冲 → TIM3_CH1(PA6) 输入捕获 Echo 上升沿和下降沿 → 脉宽/58=距离(cm)。超时 100ms 无回波返回 -1
- 实现结果：测距精度 ±1cm（2cm~200cm 范围），刷新率约 10Hz
- 关键代码：`SR04_Init()` 输入捕获配置 + `SR04_GetDistance()` 捕获中断 + 距离换算

**4.5 DHT11 温湿度实现**
- 实现方法：PA1 双向 GPIO，主机发起始信号（拉低 ≥18ms）→ 等待 DHT11 响应（80μs 低 + 80μs 高）→ 连续读 40 位数据（湿度整/小 + 温度整/小 + 校验和）→ 校验验证。位判断通过 delay 40μs 后采样电平
- 实现结果：读取成功率 > 95%，温湿度数据准确
- 关键代码：`DHT_ReadData()` 40 位读取 + 校验和验证

**4.6 红外避障传感器实现**
- 实现方法：PF5~8 上拉输入，低电平=检测到障碍，提供 `IR_LEFT()` / `IR_RIGHT()` 等宏供上层调用
- 实现结果：响应速度 < 1ms，检测距离约 5cm~15cm（传感器物理特性）
- 关键代码：`IR_Init()` GPIO 配置 + 检测宏定义

### 第四章 详细设计与实现

**4.1 开发环境介绍**
- 同通用模板，你的模块最简单，不需要额外工具

**4.2 LED 驱动实现**
- 实现方法：PB5(LED0) 和 PE5(LED1) 配置为推挽输出，初始输出高电平（LED 熄灭），低电平点亮。提供 `led_init()` 初始化函数
- 实现结果：LED 正常亮灭，可指示系统运行状态
- 关键代码：`led_init()` GPIO 初始化

**4.3 蜂鸣器驱动实现**
- 实现方法：PF0 配置为推挽输出，`beep_on(ms)` 在指定时间内以 2kHz 频率翻转 PF0 输出方波，`beep_tone(freq, ms)` 支持任意频率，`beep_sprinkler()` 播放预定义的洒水车音乐旋律序列（频率+时长数组）
- 实现结果：音调准确，音乐播放流畅
- 关键代码：`beep_tone()` 频率 → 周期换算 + delay 控制脉宽

**4.4 按键驱动实现**
- 实现方法：PE3(KEY1) 和 PE4(KEY0) 配置为上拉输入，启用 EXTI 下降沿中断。中断服务函数中通过 `delay_ms(10)` 软件消抖后置位按键状态标志，主循环中轮询标志执行对应操作
- 实现结果：按键响应灵敏，无误触发
- 关键代码：`KEY_EXTI_Init()` EXTI 配置 + `EXTI3_IRQHandler()` / `EXTI4_IRQHandler()` 中断处理

**4.5 系统初始化实现**
- 实现方法：`sys.c` 中配置 HSE 外部 8MHz 晶振 → PLL ×9 = 72MHz 系统时钟，配置 APB1=36MHz、APB2=72MHz。`NVIC_Configuration()` 设置中断优先级分组为组 2（2 位抢占 + 2 位响应）
- 实现结果：系统稳定运行在 72MHz，中断优先级正常
- 关键代码：`SystemInit()` 时钟配置 + `NVIC_PriorityGroupConfig()`

**4.6 位图资源**
- 实现方法：`bmp.h/c` 存储预定义的静态位图数组（BMP1、BMP2 等），数据采用逐列逐页格式，与 SSD1306 GDDRAM 存储格式一致，供 OLED 驱动直接调用
- 实现结果：位图显示正常
- 关键代码：位图数组定义（展示部分数据即可）
